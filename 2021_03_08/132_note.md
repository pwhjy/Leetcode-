## 132. 分割回文串Ⅱ

### 题目：

> 给你一个字符串 `s`，请你将 `s` 分割成一些子串，使每个子串都是回文。
>
> 返回符合要求的 **最少分割次数** 。

**示例1：**

```
输入：s = "aab"
输出：1
解释：只需一次分割就可将 s 分割成 ["aa","b"] 这样两个回文子串。
```

**示例2：**

```
输入：s = "a"
输出：0
```

**示例3：**

```
输入：s = "ab"
输出：1
```

**提示：**

- `1 <= s.length <= 2000`
- `s` 仅由小写英文字母组成



### 题解：

本题的标签是`动态规划`。

本题的考察重点主要有两个：

> - 设计状态转移方程计算得到最少分割次数
> - 设计状态转移方程分析字符串s中s[i,j]是否为一个回文串

第一步：

> 设f[i]为字符串s[0,i]的最少分割次数。那么f[i]=f[j]+1，这里f[j]为s[0,j]的最小分割次数。
>
> 具体的状态转移方程如下：
>
> $$f[i]=min_{i≤j≤i}\{f[j]+1\}，其中s[j+1..i]是一个回文串$$

第二步：

> 第二步需要判断s[i,j]部分是否为一个回文串。这里依旧可以采用状态转移方程来判断：
>
> - ans\[i\][j]用来表示s\[i\][j]是都是一个回文串，那么ans\[i\][j]=True(i≥j)，ans\[i\][j]=(ans\[i+1\][j-1])&&(s\[i\]==s\[j\])(i<j)
>
> 第二步的时间复杂度为O(1)。
