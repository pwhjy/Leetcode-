## 224. 基本计算器

实现一个基本的计算器来计算一个简单的字符串表达式 `s` 的值。

**示例 1：**

```
输入：s = "1 + 1"
输出：2
```

**示例 2：**

```
输入：s = " 2-1 + 2 "
输出：3
```

**示例 3：**

```
输入：s = "(1+(4+5+2)-3)+(6+8)"
输出：23
```

**提示：**

- `1 <= s.length <= 3 * 105`
- `s` 由数字、`'+'`、`'-'`、`'('`、`')'`、和 `' '` 组成
- `s` 表示一个有效的表达式



### 题解：

本题中包含了括号的匹配，那么很容易就想到栈的使用。

**注意点：**

- 由于本题可能出现负数加减的情况，所以我们设置一个sign来表示正负值。

- s中包括空格，所以在遍历字符串的时候，遇到空格跳过。
