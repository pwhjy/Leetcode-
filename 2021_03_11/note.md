## 227. 基本计算器Ⅱ

给你一个字符串表达式 `s` ，请你实现一个基本计算器来计算并返回它的值。

整数除法仅保留整数部分。

**示例 1：**

```
输入：s = "3+2*2"
输出：7
```

**示例 2：**

```
输入：s = " 3/2 "
输出：1
```

**示例 3：**

```
输入：s = " 3+5 / 2 "
输出：5
```



### 题解：

​		与224题不同的是，本题增加了`*`与`\`两个运算符号，那么我们就需要对符号两边的数字进行运算之后再添加到栈中。

​		具体思路：

​		遍历整个字符串，遇到空格跳过。由于可以从题意中读出，第一个字符肯定为数字（非操作符），所以预设一个sign，值为`+`，sign的作用是保存上一个操作符，这样遇到下一个操作符后可以进行相关的计算操作。后续中，遇到`+`，将当前计算值入栈；遇到`-`，将当前计算值取负数入栈；遇到`*`，获取栈顶数字，并与`*`后的数字相乘再入栈；同理`\`，除以`\`后的数字再入栈。
